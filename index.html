📚 このサイトについて

開発作業で頻繁に使用するコマンドや記法を素早く参照できるチートシート集です。

各ページでは、コピーボタンでコマンドを簡単にクリップボードにコピーできます。

特徴

  • ワンクリックでコードをコピー
  • ディレクトリ名やファイル名を動的に置き換え
  • プログラマーに優しいダークテーマ
  • GitHub Pagesで公開可能
