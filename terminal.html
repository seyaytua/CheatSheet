💻 ターミナルコマンド チートシート

🔄 カスタマイズ設定

ファイル・ディレクトリ操作

コマンド 説明
ls -la ファイル一覧を詳細表示
cd {DIRECTORY} ディレクトリ移動
pwd 現在位置を表示
mkdir {DIRECTORY} ディレクトリ作成
mkdir -p {DIRECTORY}/sub 親ディレクトリも含めて作成
touch {FILENAME} ファイル作成
rm {FILENAME} ファイル削除
rm -rf {DIRECTORY} ディレクトリを強制削除
cp {FILENAME} backup.txt ファイルコピー
mv {FILENAME} new.txt ファイル移動・リネーム

ファイル内容の表示・編集

ファイル内容の表示 
cat {FILENAME}
less {FILENAME}
head -n 10 {FILENAME}
tail -n 10 {FILENAME}
ファイル作成と書き込み 
cat > {FILENAME} << 'EOF'
ここに内容を書く
複数行も可能
EOF
ファイルに追記 
echo "追加する内容" >> {FILENAME}

検索・フィルタリング

ファイル内検索 
grep "検索語" {FILENAME}
grep -r "検索語" {DIRECTORY}
grep -i "検索語" {FILENAME}  # 大文字小文字を区別しない
ファイル検索 
find . -name "{FILENAME}"
find . -type f -name "*.txt"
find . -type d -name "{DIRECTORY}"

パーミッション・所有権

コマンド 説明
chmod +x {FILENAME} 実行権限を付与
chmod 755 {FILENAME} パーミッション設定 (rwxr-xr-x)
chmod -R 755 {DIRECTORY} 再帰的にパーミッション設定
chown user:group {FILENAME} 所有者変更

ショートカットキー

キー 説明
Ctrl + C 実行中断
Ctrl + D ターミナル終了 / EOF入力
Ctrl + L 画面クリア
Ctrl + R コマンド履歴検索
Tab 補完
↑ / ↓ コマンド履歴移動

プロセス管理

プロセス確認 
ps aux
ps aux | grep python
top
htop
プロセス終了 
kill PID
kill -9 PID  # 強制終了
killall process_name
